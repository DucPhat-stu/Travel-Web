# Module Assignment

## Member 1 (Backend Auth)
- Thư mục: users/, travel_tourism/settings/
- Chức năng:
  + Đăng ký, đăng nhập, đăng xuất
  + Quên mật khẩu / reset mật khẩu
  + Phân quyền user/admin
  + Redirect sau login
- Giai đoạn hoàn thành: Tuần 1

## Member 2 (Frontend UI/UX)
- Thư mục: templates/, static/, admin_panel/templates/
- Chức năng:
  + Thiết kế layout base
  + Giao diện user (homepage, tour, hotel, flight)
  + Giao diện admin dashboard
- Giai đoạn hoàn thành: Tuần 1 → Tuần 2

## Member 3 (Database + Booking)
- Thư mục: tours/, hotels/, flight/, bookings/
- Chức năng:
  + Tạo models cho Tour, Hotel, Flight
  + Tạo model Booking liên kết user và dịch vụ
  + CRUD dữ liệu
- Giai đoạn hoàn thành: Tuần 2

## Member 4 (Chatbot + Logs)
- Thư mục: chatbot/, data/, scripts/
- Chức năng:
  + Chuẩn bị intents.json
  + Train model chatbot
  + Tích hợp giao diện chatbot vào user site
  + Hiển thị lịch sử chat trong admin panel
- Giai đoạn hoàn thành: Tuần 3
